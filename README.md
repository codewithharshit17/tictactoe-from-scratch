# Tic Tac Toe

A complete Tic Tac Toe game created from scratch using basic HTML, CSS, and JavaScript.

This is a beginner-friendly browser game where two players take turns placing `O` and `X` marks on a 3x3 board. The game checks for a winner, shows the winning message, disables the board after the game is won, and allows the player to reset the game.

## Features

- Built with plain HTML, CSS, and JavaScript
- Interactive 3x3 Tic Tac Toe board
- Two-player turn system for `O` and `X`
- Boxes are disabled after being clicked once
- Winner detection using all possible winning patterns
- Winning message displayed on the screen
- Board is disabled after a player wins
- Reset button to restart the game
- Simple and clean styling

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Files

- `index.html` contains the main page structure
- `style.css` contains the styling for the game
- `app.js` contains the game logic

## How to Run

1. Download or clone this project.
2. Open the project folder.
3. Open `index.html` in any web browser.
4. Start playing Tic Tac Toe.

## About This Project

I created this Tic Tac Toe game from scratch while learning the basics of web development. The goal of this project is to practice DOM manipulation, event handling, button states, functions, arrays, and basic game logic using JavaScript.

## Game Rules

1. Player `O` starts the game.
2. Players take turns clicking empty boxes.
3. A clicked box cannot be used again.
4. The first player to match three marks in a row, column, or diagonal wins.
5. After a player wins, the board is disabled.
6. The reset button clears the board and starts a new game.
