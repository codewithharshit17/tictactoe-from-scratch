# Tic Tac Toe

A simple Tic Tac Toe game created from scratch using basic HTML, CSS, and JavaScript.

This project is a beginner-friendly browser game where two players can take turns placing `O` and `X` marks on a 3x3 board. Each box can be clicked only once, just like a real Tic Tac Toe board.

## Features

- Built with plain HTML, CSS, and JavaScript
- Interactive 3x3 Tic Tac Toe board
- Two-player turn system for `O` and `X`
- Boxes are disabled after being clicked once
- Reset button included in the interface
- Simple and clean styling

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Files

- `index.html` contains the main page structure
- `style.css` contains the styling for the game
- `app.js` contains the game logic

## How to Run

1. Download or clone this project.
2. Open the project folder.
3. Open `index.html` in any web browser.
4. Start playing Tic Tac Toe.

## About This Project

I created this Tic Tac Toe game from scratch while learning the basics of web development. The goal of this project is to practice DOM manipulation, event handling, button states, and basic game logic using JavaScript.
